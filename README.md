# Design Lab — AndDream 设计实验室

AndDream 品牌的设计实验中心。收录 4 个已完成品牌页面版本 + 10 个待做前沿设计实验。

---

## 工作流程

每个实验必须严格遵循以下流程，不可跳过任何步骤：

```
① 背景介绍 → ② 需求沟通 → ③ 设计方案 → ④ 逐步实现 → ⑤ 完成挂入画廊
```

| 步骤 | 说明 |
|------|------|
| **① 背景介绍** | 介绍该设计方向是什么、从哪里来、有哪些代表案例，让 Andrew 充分了解背景后再开始 |
| **② 需求沟通** | 了解 Andrew 的想法、偏好、具体需求，基于实际需求设计，不做假设 |
| **③ 设计方案** | 提出具体设计方案供 Andrew 确认，确认后再动手 |
| **④ 逐步实现** | 一步一步实现，每步确认无误再继续 |
| **⑤ 完成挂入** | 完成后更新总览画廊 `index.html`，将对应卡片状态改为已完成 |

> ⚠️ 禁止跳过任何步骤直接写代码。
