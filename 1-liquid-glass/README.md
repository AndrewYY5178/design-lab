# AndDream Liquid Glass — 液态玻璃品牌页

基于 CSS `backdrop-filter` 的液态玻璃风格品牌页面。浅色画作背景 + 极透毛玻璃卡片，AndDream 品牌完整内容分 10 张卡片呈现。

---

## 预览

- 定制画作全屏背景
- 10 张毛玻璃卡片承载品牌内容
- 鼠标悬停高光追踪
- 滚动视差效果
- 纯 CSS + 少量 JS，无框架

## 打开方式

直接用浏览器打开 `index.html`。

> 需要联网加载 Google Fonts。

## 技术栈

- CSS `backdrop-filter: blur()` + `saturate()`
- CSS 渐变镜面高光
- JavaScript 鼠标追踪 + 滚动视差
